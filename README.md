# Fitness Data API & Dashboard

A full-stack fitness application that serves exercise data via a RESTful API and displays it on an interactive dashboard.

## Features
* **REST API:** Built with Java Spring Boot to handle HTTP requests.
* **Database:** PostgreSQL database storing complex exercise metadata (muscle groups, equipment, difficulty).
* **Smart Filtering:** Filter exercises by muscle group (e.g., "Abs"), difficulty ("Beginner"), or equipment.
* **Frontend Dashboard:** Interactive HTML/JS interface with dynamic search and card-based layout.

## Tech Stack
* **Backend:** Java 17, Spring Boot, Hibernate, Spring Data JPA
* **Database:** PostgreSQL
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)
* **Tools:** Maven, IntelliJ IDEA, Postman

## How to Run
1.  Clone the repository: `git clone https://github.com/das622/Fitness-Data-App.git`
2.  Configure `application.properties` with your PostgreSQL credentials.
3.  Run the Spring Boot application.
4.  Open `gym-frontend/index.html` in your browser.
